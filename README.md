# ArrowOS
A miniOS designed on Visual Studios with Cosmos 

![Front](https://github.com/deepakshisud/ArrowOS/blob/master/Images/StartPage.png)

**Author**
Deepakshi Sood :grin:

----------------------------------------------------------------------------

**Features:**

* Basic Commands
 * Shutdown
 * Clear
 * Reboot
 * Lock System
 * Help
 * Date & Time
 
* File Functionalities
 * Create Directory
 * List all Directories and Files
 * Delete Directory
 * Open File
 * Copy File
 * Concatenate Files
 * Search Files
 * Recycle Bin
 
* Calculator

* Software Applications
 * Text Editor
 * Snake Game
